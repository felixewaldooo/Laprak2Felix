1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web ?

CSS digunakan dalam pemrograman web untuk membuat halaman web lebih menarik dan terorganisir dengan mengatur tata letak, warna, ukuran font, dan gaya visual lainnya. Ini memungkinkan elemen halaman untuk tampil lebih rapi dan konsisten di berbagai perangkat, meningkatkan pengalaman pengguna secara keseluruhan. JavaScript digunakan dalam pemrograman web untuk untuk memberikan kemampuan interaktif seperti animasi, validasi form, dan respons dinamis, yang membuat pengalaman pengguna lebih hidup dan responsif.

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web ?

CSS dapat diintegrasikan dalam pemrograman web dengan menulis langsung kode CSS pada atribut elemen HTML. Kode internal CSS ditulis dalam tag "style" di bagian "head" file HTML, dan kode eksternal CSS ditulis dalam file.css terpisah yang dihubungkan dengan elemen "rel="stylesheet" href="style.css." Karena membuat struktur HTML dan gaya visualnya lebih mudah dikelola, eksternal CSS biasanya lebih sering digunakan. JavaScript dapat diintegrasikan dalam pemrograman web dengan menulis langsung pada elemen HTML menggunakan atribut event handler, seperti onclick, onmouseover, atau onchange. Misalnya, button onclick="alert('Hello!')" Klik saya /button akan menampilkan pesan saat tombol diklik. Atau, dengan memasukkannya dalam tag script di dalam dokumen HTML, biasanya di bagian head atau di akhir bagian body. Penempatan di akhir body untuk mempercepat pemuatan halaman, karena JavaScript akan dieksekusi setelah elemen-elemen HTML dimuat.

3. Jelaskan apa saja keuntungan menggunakan JavaScipt dan CSS ?

Keuntungan menggunakan JavaScript : 1. JavaScript memungkinkan halaman web menjadi lebih interaktif dengan memungkinkan pengguna berinteraksi dengan halaman secara langsung, seperti ketika mereka mengklik tombol atau mengarahkan kursor ke elemen tertentu (hover). JavaScript dapat melakukan hal-hal seperti menampilkan pop-up atau memperluas informasi tanpa memuat ulang halaman. 2. Pengembang dapat mengubah konten dan tampilan halaman secara dinamis tanpa memuat ulang seluruh halaman dengan menggunakan JavaScript, yang dapat menambah, menghapus, atau mengubah elemen HTML, seperti menampilkan pesan baru, mengubah gambar, atau mengubah daftar produk. 3. JavaScript memiliki berbagai alat bantu, seperti React dan Vue, yang dimaksudkan untuk mempermudah pengembangan aplikasi web yang lebih kompleks dan interaktif. React memungkinkan pembuatan antarmuka pengguna (UI) dinamis dengan menggunakan komponen yang dapat digunakan kembali, sedangkan Vue menyediakan framework ringan yang memungkinkan integrasi yang mudah. Keuntungan menggunakan CSS : 1. Mengelola tampilan secara terpisah dari struktur halaman, sehingga HTML tetap bersih dan mudah dibaca. CSS mengatur elemen visual dan tata letak secara terpisah dari struktur HTML. Hasilnya, file HTML menjadi lebih teratur dan mudah dilihat, yang memudahkan pengembang untuk mengedit kode. 2. Memungkinkan situs untuk menyesuaikan tampilannya di berbagai perangkat dengan menggunakan media query, sehingga situs lebih responsif. Pengembang dapat menggunakan fitur media query untuk menyesuaikan tampilan halaman berdasarkan ukuran layar atau perangkat yang digunakan pengguna. 3. Memberikan efek animasi dan transisi tanpa JavaScript tambahan. CSS memungkinkan pengalaman yang lebih halus dan dinamis tanpa menggunakan JavaScript dengan efek animasi dan transisi yang mudah.

4. Jelaskan apa saja keuntungan menggunakan Bootstrap ?

1. Bootstrap menyediakan framework CSS dan JavaScript siap pakai yang mempermudah pembuatan desain responsif dan tampilan yang konsisten di berbagai perangkat. 2. Dengan grid sistemnya, Bootstrap memungkinkan pembuatan halaman web yang otomatis menyesuaikan dengan ukuran layar perangkat, seperti tablet, ponsel, desktop, atau tablet. 3. Bootstrap menyediakan komponen antarmuka pengguna, seperti formulir, tombol, navigasi, dan modals, yang dapat digunakan langsung untuk mempercepat pengecekan. 4. Menjamin bahwa tampilan situs tetap konsisten karena semua komponen menggunakan desain dan aturan yang seragam.5. Bootstrap dilengkapi dengan dokumentasi yang jelas dan contoh kode, yang memudahkan pengembang pemula maupun berpengalaman dalam mengimplementasikan fitur framework ini.
